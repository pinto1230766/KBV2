# KBV2 - Application Web Locale

## Installation et Démarrage Automatique

### 1. Démarrage Manuel
Double-cliquez sur `start-kbv2.bat` pour lancer l'application web.

### 2. Démarrage Automatique au démarrage du PC
1. Appuyez sur `Win + R`
2. Tapez `shell:startup` et appuyez sur Entrée
3. Copiez `auto-start-kbv2.bat` dans ce dossier
4. Redémarrez votre PC pour tester

### 3. Accès à l'application
- URL: http://localhost:5173
- L'application démarre automatiquement 30 secondes après le démarrage du PC

## Synchronisation WhatsApp

### Sauvegarde depuis le PC
1. Allez dans Paramètres > Sauvegarde
2. Cliquez sur "Créer une sauvegarde"
3. Le fichier sera généré et partagé via WhatsApp

### Restauration sur Mobile
1. Recevez le fichier de sauvegarde via WhatsApp
2. Sur mobile/tablette, ouvrez le fichier dans KBV2
3. Les données seront automatiquement importées

## Dépannage

Si l'application ne démarre pas:
1. Vérifiez que Node.js est installé
2. Ouvrez une invite de commande et exécutez `node --version`
3. Si nécessaire, installez Node.js depuis https://nodejs.org
