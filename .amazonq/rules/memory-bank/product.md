# Product Overview - KBV Lyon

## Project Purpose
KBV Lyon is a comprehensive visitor speaker management application developed for the Baptist Church of Lyon (Ã‰glise Baptiste de Lyon). It streamlines the management of visiting speakers, their visits, hosting logistics, and communication workflows.

## Value Proposition
- **Centralized Management**: Single platform for all speaker-related operations
- **Production-Ready Architecture**: Enterprise-grade security, performance, and accessibility
- **Mobile-First Design**: Optimized for Samsung Tab S10 Ultra and mobile devices
- **Offline Capability**: Full functionality with background synchronization
- **Multi-Language Support**: French, Portuguese, and Cape Verdean Creole (KEA)

## Key Features

### Core Functionality
- **Speaker Management**: Complete CRUD operations with detailed profiles, contact information, and visit history
- **Host Management**: Accommodation system with host profiles and availability tracking
- **Visit Planning**: Intelligent calendar with conflict detection and automated scheduling
- **Dashboard Analytics**: Real-time KPIs, interactive charts, and trend analysis
- **Communication System**: Message templates, history tracking, and automated notifications
- **Expense Tracking**: Financial management for speaker-related costs
- **Report Generation**: Comprehensive reporting with export capabilities

### Technical Excellence
- **Security (100%)**: JWT authentication, AES-GCM encryption, Zod validation, XSS protection
- **Performance (80%)**: List virtualization, intelligent caching, lazy loading, bundle splitting
- **Accessibility (70%)**: WCAG compliance, keyboard navigation, ARIA attributes, screen reader support
- **Testing (85%)**: Unit tests (Vitest), E2E tests (Playwright), component documentation (Storybook)

### Mobile & Platform Features
- **Touch Gestures**: Swipe, pinch, long press interactions
- **Offline Mode**: IndexedDB storage with sync queue
- **Pull-to-Refresh**: Native mobile experience
- **Samsung Optimizations**: Device-specific adaptations for Tab S10 Ultra
- **File System Integration**: Capacitor-based file management for Android/iOS

## Target Users
- **Church Administrators**: Manage speakers, visits, and logistics
- **Coordinators**: Plan schedules and handle communications
- **Hosts**: View assignments and speaker information
- **Leadership**: Access analytics and reports

## Use Cases
1. **Speaker Onboarding**: Add new speakers with complete profiles and preferences
2. **Visit Scheduling**: Plan visits with automatic conflict detection and host assignment
3. **Communication**: Send templated messages for invitations, confirmations, and follow-ups
4. **Analytics**: Track visit trends, speaker frequency, and expense patterns
5. **Reporting**: Generate reports for leadership and record-keeping
6. **Mobile Access**: Manage operations on-the-go with offline support
7. **Data Backup**: Export/import data with Samsung Documents integration
